The weather_app module¶

The weather_app module provides a high-level interface to the SMHI forecast. The module currently provides function to access the temperature and the precipitation forecast data. Accessing the weather forecast requires an active internet connection. The forecast location is determined automatically based on location lookup of the internet access point.

Since SMHI doesn’t provide weather forecasts outside of Sweden through their web API, request from outside of Sweden use Gothenburg as location.

weather_app.get_precipitation_forecast()¶

Retrieve current SMHI predicted precipitation .

Linearly interpolates the current SMHI precipitation forecast to the given array of lead times. Precipitation here includes rain, snow, graupel or hail.

Parameters

hours (array) – 1D array containing the lead times in hours for which to compute the precipitation.

Returns

Array containing the predicted temperatures.

weather_app.get_temperature_forecast(hours)¶

Retrieve current SMHI predicted temperatures.

Linearly interpolates the current SMHI temperature forecast to the given array of lead times.

Parameters

hours (array) – 1D array containing the lead times in hours for which to compute the temperature.

Returns

Array containing the predicted temperatures.
