The weather_app.render module¶

The wether_app.render module provides functionality to render weather forecast data as ASCII plots. All functionality is implemented by the AsciiGraph class.

class weather_app.render.AsciiGraph(height=40, color=True)¶

Drawing graphs as ASCII string.

The AsciiGraph class represents a graph to be rendered to a string using ASCII characters.

height¶

The height of the plot in lines.

Type

int

color¶

Whether or not to use BASH color codes in output.

Type

Boolean

grid¶

Numpy array containing the characters for each position in the panel.

Type

np.array

I_ZERO¶

Grid index corresponding to 0 on the y-axis.

I_MAX¶

Grid index corresponding the maximum y-value.

J_ZERO¶

Grid index corresponding to 0 on the x-axis.

J_MAX¶

Grid index corresponding the maximum x-value.

add_precipitation_plot(x, y)¶

Add precipitation plot to the graph.

Draws the given precipitation sequence into the plot.

Parameters

  • x (array) – 1D array containing the x coordinates of the sequence.

  • y (array) – 1D array containing the corresponding precipitation.

add_temperature_plot(x, y)¶

Add temperature plot to the graph.

Draws the given temperature sequence into the plot.

Parameters

  • x (array) – 1D array containing the x coordinates of the sequence.

  • y (array) – 1D array containing the corresponding temperatures.

property precipitation_marker¶

The marker (including color codes) used to draw precipitation.

render()¶

Prints the resulting graph to the standard out.

set_time_range(t0, t1)¶

Set the x-axis range of the plot.

This draws x-labels onto the plot according the given start and end times of the plotted sequences.

Parameters

  • t0 (datetime) – The time to associate with the leftmost column of the plot.

  • t1 (datetime) – The time to associate with the rightmost column of the plot.
